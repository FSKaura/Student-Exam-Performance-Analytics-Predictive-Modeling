# Student-Exam-Performance-Analytics-Predictive-Modeling
This repository contains an end-to-end data analytics project focused on identifying key factors influencing student academic success. By analyzing behavior variables like study habits, sleep patterns, and historical grades, we developed insights to help educational institutions implement early-intervention strategies for at-risk students.

## 📊 Brief Dataset Description
The dataset analyzed in this project consists of 200 distinct student records containing information on their academic environments and behaviors. The features included are:

* **`student_id`**: A unique identifier assigned to each student.
* **`hours_studied`**: The total number of hours a student dedicated to independent study per week.
* **`sleep_hours`**: The average number of hours of sleep a student gets per night.
* **`attendance_percent`**: The student's overall classroom attendance rate (%).
* **`previous_scores`**: The historical average or past grades achieved by the student.
* **`exam_score`**: The final target variable representing the student's final exam performance.
